# Grug-MoE v4 path standardized effect-size plots

These plots summarize the one-dimensional path from proportional Grug-MoE to v4. Deltas are oriented so positive is better, then divided by the empirical per-task standard deviation of oriented metric values over available Grug-MoE dashboard/path cells. This is an effect-size diagnostic, not a repeated-seed noise standard deviation.
