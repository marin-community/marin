# Source info — legacy-independent scenarios

## Generation method

These scenarios were generated by a single GPT-5.1 call per statement.

- **Model**: gpt-5.1
- **Temperature**: 0
- **reasoning_effort**: "none"
- **response_format**: `{{"type": "json_object"}}`
- **Generator script**: `experiments/posttrain/disagreement_primitive/e8_paired_indirection.py::stage2_generate_scenarios` (line ~348)
- **System prompt**: `SCENARIO_GEN_SYSTEM` constant in the same file (line ~109)
- **Per-statement prompt**: `"STATEMENT TEXT:\n{{statement_text}}\n\nGenerate 20 borderline scenarios per the schema."`

## Shape of each scenario

Each scenario is a JSON object with a single field:
```json
{{"user_query": "<the user message>"}}
```

There is no explicit axis structure attached to individual scenarios. The
system prompt asks the LM to "Generate exactly 20 distinct scenarios" that
are "borderline" and "span different angles (different user types,
contexts, framings) — not paraphrases of one base case." Diversity is
internal-to-the-LM (single-call context window) rather than enforced by
explicit axis sampling.

## Volume

20 scenarios per statement.
