# MCT-LRQ Mixture-Term Ablation

Date: 2026-04-24

## Question

How much of barrier-free MCT-LRQ's predictive quality comes from actual smooth mixture-dependent terms, versus only scale continuation?

## Variants

- `mct_lrq69_full_no_barrier`: current barrier-free MCT-LRQ predictive law, with LRQ mixture anchor and family-dependent D scale head.
- `mct_lrq65_anchor_global_scale`: same LRQ mixture anchor, but A/B/C are all global constants. This removes mixture-dependent scale heads but keeps the mixture regression anchor.
- `identity_anchor_global_scale`: lookup-table transfer from observed 100M/6B mixture IDs plus global scale terms. This learns no smooth mixture features; it only memorizes identities where available.
- `identity_anchor_chinchilla_approach3`: same identity lookup-table anchor, but with strict two-term Approach 3 scaling and fitted global `alpha,beta`.
- `scale_only_global`: one global scaling curve with no mixture information.
- `scale_only_chinchilla_approach3`: strict pure-scale Chinchilla Approach 3 control: `E + A*uN + B*uD`, with fitted global `alpha,beta` and no MCT cross term.

## Main Result

The scale-only controls are not competitive. The identity-transfer control is useful for seen mixtures but does not solve generalization, and it cannot define meaningful raw-simplex optima because custom mixtures fall back to the mean anchor. The gap between `mct_lrq69_full_no_barrier` and `mct_lrq65_anchor_global_scale` measures the value of mixture-dependent scale heads; the much larger gap to the pure-scale controls measures the value of the LRQ/GRP-style mixture anchor.
