Plan: Reproduce NemotronTerminal-8B on Terminal-Bench 2.0 (13.0 ± 2.2)                              │
│                                                                                                     │
│ Context                                                                                             │
│                                                                                                     │
│ Issue https://github.com/marin-community/marin/issues/3490 asks us to reproduce the eval results of │
│  nvidia/Nemotron-Terminal-8B on Terminal-Bench 2.0 (TB), which achieves 13.0 ± 2.2 in the           │
│ https://arxiv.org/abs/2602.21193.                                                                   │
│                                                                                                     │
│ Key facts from the paper:                                                                           │
│ - Model: nvidia/Nemotron-Terminal-8B (SFT from Qwen3-8B)                                            │
│ - Agent: Terminus 2 (model-agnostic reference agent from Terminal-Bench 2.0)                        │
│ - Benchmark: Terminal-Bench 2.0 (89 tasks)                                                          │
│ - Environment: Daytona (cloud sandboxes)                                                            │
│ - Eval context window: 40,960 tokens (default Qwen3-8B settings)                                    │
│ - The ± 2.2 is binomial standard error over 89 binary-outcome tasks                                 │
│                                                                                                     │
│ Steps                                                                                               │
│                                                                                                     │
│ Step 1: Download model weights to GCS                                                               │
│                                                                                                     │
│ Download nvidia/Nemotron-Terminal-8B from HuggingFace to GCS following the existing naming          │
│ convention.                                                                                         │
│                                                                                                     │
│ # On a machine with HF access and gcloud auth                                                       │
│ pip install huggingface_hub                                                                         │
│ python -c "                                                                                         │
│ from huggingface_hub import snapshot_download                                                       │
│ snapshot_download('nvidia/Nemotron-Terminal-8B', local_dir='/tmp/Nemotron-Terminal-8B')             │
│ "                                                                                                   │
│ gcloud storage cp -r /tmp/Nemotron-Terminal-8B                                                      │
│ gs://marin-us-central1/models/nvidia--Nemotron-Terminal-8B--main/                                   │
│                                                                                                     │
│ Target path: gs://marin-us-central1/models/nvidia--Nemotron-Terminal-8B--main/                      │
│                                                                                                     │
│ Step 2: Create experiment file                                                                      │
│                                                                                                     │
│ Create experiments/exp3490_nemotron_terminal_8b_tb.py modeled on the existing                       │
│ experiments/exp2602_harbor_ot_tb_dev.py.                                                            │
│                                                                                                     │
│ Key configuration:                                                                                  │
│ - DATASET = "terminal-bench" (from Harbor registry, not HF)                                         │
│ - VERSION = "2.0"                                                                                   │
│ - HARBOR_AGENT = "terminus-2"                                                                       │
│ - MODEL_NAME = "nemotron-terminal-8b"                                                               │
│ - MODEL_PATH = "gs://marin-us-central1/models/nvidia--Nemotron-Terminal-8B--main/"                  │
│ - ENV_TYPE = "daytona"                                                                              │
│ - HARBOR_N_CONCURRENT = 25 (reasonable parallelism)                                                 │
│ - HARBOR_MAX_INSTANCES: defaults to None (all 89 tasks), overridable via env var for smoke test     │
│ - Resource: ResourceConfig.with_tpu("v5p-8") (same as exp2602)                                      │
│                                                                                                     │
│ Critical files to reference:                                                                        │
│ - Template: experiments/exp2602_harbor_ot_tb_dev.py                                                 │
│ - Eval function: experiments/evals/evals.py → evaluate_harbor()                                     │
│ - Harbor evaluator: lib/marin/src/marin/evaluation/evaluators/harbor_evaluator.py                   │
│                                                                                                     │
│ Step 3: Smoke test (5 tasks)                                                                        │
│                                                                                                     │
│ Run with HARBOR_MAX_INSTANCES=5 to validate the setup:                                              │
│                                                                                                     │
│ uv run lib/marin/src/marin/run/ray_run.py \                                                         │
│     --env_vars MARIN_PREFIX gs://marin-us-central1 \                                                │
│     --env_vars DAYTONA_API_KEY ${DAYTONA_API_KEY} \                                                 │
│     --env_vars WANDB_API_KEY ${WANDB_API_KEY} \                                                     │
│     --env_vars WANDB_ENTITY marin-community \                                                       │
│     --env_vars WANDB_PROJECT harbor \                                                               │
│     --env_vars HF_TOKEN ${HF_TOKEN} \                                                               │
│     --env_vars HARBOR_MAX_INSTANCES 5 \                                                             │
│     --cluster us-central1 \                                                                         │
│     --extra harbor,vllm \                                                                           │
│     --no_wait \                                                                                     │
│     -- python experiments/exp3490_nemotron_terminal_8b_tb.py                                        │
│                                                                                                     │
│ Step 4: Full run (all 89 tasks)                                                                     │
│                                                                                                     │
│ Once smoke test passes, run with all tasks (no HARBOR_MAX_INSTANCES override):                      │
│                                                                                                     │

│     --extra harbor,vllm \                                                                           │
